Procedure to Create and View a Website on the Production Server

  1. Open your terminal and make sure you are in your folder
  2. Type in "sftp authur2718@cmis-hsteacher"
  3. When the question jumps up, type in "yes". Then type in your password.
  4. After Sftp have shown in your position text, type "cd public_html/"
  5. To add stuff in or to update, type "put filename" to put your files in.
  6. To view your context, type in the web browser "html://authur2718@cmis-hsteacher". If the server is open, you can view your conext.
